# GitBook 安装

## 一 安装 Node.js

1 Node.js 官网

<https://nodejs.org/en/>

<http://nodejs.cn/>

2 安装教程

<https://www.runoob.com/nodejs/nodejs-install-setup.html>

3 检测是否安装成功

````
node -v
````

## 二 安装 GitBook

1 安装命令

````
npm install gitbook-cli -g
````

2 检测是否安装成功

````
gitbook -V
````
